Workshop Information

Below, you will be able to sign up for our Summer 2023 workshops. This summer, each class will be held virtually using Zoom. Each workshop is held on a different week, so we encourage you to register for as many as you want! Note: There is one registration form for all the classes; there isn't an individual form for each.

The best part? Everything's free!

Available workshops this summer:

Intro to Java

Duration: 08/21/2023 - 08/25/2023

Time: Every Day @ 5:00-7:00 PM (EST)

Students are introduced to the exciting world of Java and explore object-oriented programming, structured application design, and collaborative application development. Recommended for rising 5th graders and above.

Intro to Python

Duration: 07/17/2023 - 7/21/2023

Time: Every Wednesday @ 3:00-4:30PM (EST)

Students are familiarized with the fundamentals of the Python programming language and learn to design engaging, organized application projects to implement their skills. Recommended for rising 5th graders and above.

Comprehensive Web Development

Duration: 07/10/2023 - 07/14/2023

Time: Every Day @ 4:00-6:00 PM (EST)

Students explore the world of web development by learning the basics of HTML, CSS, and JavaScript. Students will leave this course with their own website and intermediate-level knowledge of the aforementioned languages. Recommended for 5th graders and above; no prior experience with coding is needed.

Scratch

Duration: 06/26/2023 - 06/30/2023

Time: Every Day @ 3:00-5:00 PM (EST)

Students explore the world of programming with fun projects and interactive activities created with Scratch. They will have fun playing and developing their own games, all while learning the basic principles of coding. Recommended for elementary and middle schoolers.

AP Computer Science Principles Prep

Duration: 08/14/2023 - 08/18/2023

Time: Every Day @ 5:00-7:00 PM (EST)

Students achieve a basic understanding of the high school Advanced Placement course Computer Science Principles, going through the syllabus and learning concepts as they go along. Disclaimer: This workshop is not endorsed by College Board, and will not fully prepare students for the AP exam; it is only meant as a preparation for the class. Recommended for rising 8th graders and above.
