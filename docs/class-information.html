Workshop Information

Below, you will be able to sign up for our Winter 2024 workshops. This winter, each class will be held virtually using Zoom. Each workshop is held on a different day, so we encourage you to register for as many as you want! Note: There is one registration form for all the classes; there isn't an individual form for each.

The best part? Everything's free!

Available workshops this winter:

Intro to Java

Duration: 01/10/2024 - 03/13/2024

Time: Every Wednesday @ 7:00-8:00 PM (EST)

Students are introduced to the exciting world of Java and explore object-oriented programming, structured application design, and collaborative application development. Recommended for 5th graders and above.

Intro to Python

Duration: 01/08/2024 - 03/11/2024

Time: Every Monday @ 7:00-8:00PM (EST)

Students are familiarized with the fundamentals of the Python programming language and learn to design engaging, organized application projects to implement their skills. Recommended for 5th graders and above.

Intro to Web Development

Duration: 01/11/2024 - 03/14/2024

Time: Every Thursday @ 7:00-8:00 PM (EST)

Students explore the world of web development by learning the basics of HTML and CSS. Students will leave this course with their own website and intermediate-level knowledge of the aforementioned languages. Recommended for 5th graders and above; no prior experience with coding is needed.

Scratch

Duration: 01/13/2024 - 03/16/2024

Time: Saturdays @ 5:00-6:00 PM (EST)

Students explore the world of programming with fun projects and interactive activities created with Scratch. They will have fun playing and developing their own games, all while learning the basic principles of coding. Recommended for elementary and middle schoolers.

AP Computer Science Principles Prep

Duration: 01/09/2024 - 03/12/2024

Time: Tuesdays @ 7:00-8:00 PM (EST)

Students achieve a basic understanding of the high school Advanced Placement course Computer Science Principles, going through the syllabus and learning concepts as they go along. Disclaimer: This workshop is not endorsed by College Board, and will not fully prepare students for the AP exam; it is only meant as a preparation for the class. Recommended for 8th graders and above.

Intro to JavaScript

Duration: 03/14/2024 - 04/11/2024

Time: Thursdays @ 7:00-8:00 PM (EST)

Students will learn the basics of JavaScript, especially with regard to its applications in web development. It is recommended that students who take Intro to Web Development to enroll in this workshop, as they will learn to add greater functionality to the websites they have created using the Document Object Model. However, Intro to Web Dev is not a prerequisite for this workshop. Recommended for 5th graders and above.
