Why join us?

KidCoders provides teachers with all the resources necessary to host their own programming classes, including:

  • Virtual and in-person learning environments
  • A classroom platform to run classes, including:
  • Announcements
  • Class material
  • Practice assignments
  • Applicative projects
  • Help with the creation of class material
  • A fun learning environment for all!
