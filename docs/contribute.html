Why join us?

KidCoders provides teachers with all the resources necessary to host their own programming classes, including:

  • Virtual and in-person learning environments
  • A classroom platform to run classes, including:
  • Announcements
  • Class material
  • Practice assignments
  • Applicative projects
  • Help with the creation of class material
  • A fun learning environment for all!

Positions

Teacher

This position is for those passionate about programming, regardless of the language(s), and are equally passionate about helping others develop those same skills.

Social Media Coordinator

This position is for those who have skills with managing social media accounts. The social media coordinator helps with growing KidCoders' Instagram account and creating/designing posts.

Volunteer

This position is for those who may not have enough time to dedicate to teaching or managing the social media accounts, but would still like to help with KidCoders' endeavors. Volunteers act as backup.
