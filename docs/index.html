KidCoders 365.

Interested in learning the various coding languages of the future? Look no further than KidCoders 365. Run for students, by students. All for free. Come join us in the Rockville Memorial Library's second floor computer lab or online via zoom. Registration for fall 2022 workshops is opening soon!

Who are we?

KidCoders 365 is an organization that offers free, interactive programmingclasses for elementary, middle, and high school students, run by high schoolstudents in Maryland.

Most popular workshops

Introductory Java

Start Date: MM/DD/YYYY

Students are introduced to the exciting world of Java and explore object-oriented programming, structured application design, and collaborative application development. Recommended for rising 5th graders and above.

Introductory Python

Start Date: MM/DD/YYYY

Students are familiarized with the fundamentals of the Python programming language and learn to design engaging, organized application projects to implement their skills. Recommended for rising 5th graders and above.

Intermediate Python

Start Date: MM/DD/YYYY

Students dive deeper into the Python programming language to explore advanced computer science concepts and develop complex, meaningful applications. Recommended knowledge of Python up to all topics covered in the beginner curriculum.

Comprehensive Web Development

Start Date: MM/DD/YYYY

Students explore the world of web development through learning the basics of HTML, CSS, and JavaScript. Students will leave this course with their own website and intermediate-level knowledge of the aforementioned languages. Recommended for 5th graders and above; no prior experience with coding is needed.
